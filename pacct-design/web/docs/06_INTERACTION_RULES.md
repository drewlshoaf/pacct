# Interaction Rules

## Run Defaults

-   Remember last selected metric
-   Remember comparison state
-   Remember AI panel state

## Comparison

-   Opens modal selector
-   No auto-compare
-   Solid vs dashed overlay lines

## Event Click

-   Zoom ±60--120s
-   Open event detail context
-   Reset zoom visible

## Density Philosophy

Balanced: - No overload - No excessive whitespace - Max 3 major visual
zones above fold
