# AI Panel Specification

AI is structured analysis, not chat.

## Entry

-   Global trigger in header
-   Contextual triggers inside Run & Compare
-   Dockable / Undockable
-   Remembers last state

## Sections

-   Primary Finding
-   Secondary Observations
-   Risk Assessment
-   Recommendations
-   Confidence
-   Evidence Links

AI must anchor findings to real events. No hallucinated causality.
